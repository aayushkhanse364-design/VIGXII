# Quick Start - VIGXII Visuals Frontend

## Install & Run

```bash
# Install dependencies
cd frontend
npm install

# Start development server
npm run dev
```

## Access

- **Public Portfolio**: http://localhost:5173
- **Admin Panel**: http://localhost:5173/admin
